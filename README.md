## Usage

```
{
  "hipchat": {
    "apikey": "1234abcdefg1234abcdefg",
    "apiversion": "v1",
    "room": "Ops",
    "from": "Sensu",
    "message_template": "optional message template erb file path - /some/path/to/template.erb"
  }
}
```
